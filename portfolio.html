PORTFOLIO

Here is a game I made in Scratch
Here's a two person tic tac toe game i made on MIT App Inventor.
At first it didn't work, but we redid the coding and fixed the problem. The issue was in the part of the program determining when to change from X's to O's turn.
This project could be made better by adding a score for how many times each player has won and adding a nice background.
If this were an actual product, you wouldn't want to buy it because the app store has better games.
The game works by setting conditions for victory in the "has won" procedure.
Multiple procedures were made to ensure that the letters appear and alternate with each turn.
